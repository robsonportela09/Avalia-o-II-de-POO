# ATM, Sistema de caixa eletrônico com Python

Projeto desenvolvido para a disciplina de Programação Orientada a Objetos(POO), ministrada pelo Professor Evandro Rodrigues.

Esse projeto de ATM basicamente, simula o funcionamento básico de um caixa eletrônico, onde permite que o usuário realize operações bancárias através do terminal, como foi proposto. Os conceitos utilizados nesse projeto estão firmados logo abaixo.

## Funcionalidades conforme pedido no GUIA em PDF

- Criar conta
- Depositar dinheiro
- Sacar dinheiro
- Consultar saldo
- Exibir histórico de operações

## Conceitos de POO utilizados nesse PJ

- Classes e Objetos
- Herança
- Polimorfismo
- Encapsulamento
- Agregação
- Composição
- Pacotes
